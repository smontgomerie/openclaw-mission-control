# Portfolio review pipeline

Morning portfolio review is generated by Mission Control from Google Sheet rows (source of truth) plus live `yfinance` enrichment, then written to the shared OpenClaw workspace under `portfolio/`.

## Flow

1. **Sheets → Mission Control** — `gog` reads `Positions` and `Trades`, then a thin script POSTs `positions_rows` and `trades_rows` to `POST /api/v1/portfolio/review/run` (org admin).
2. **Engine** — [`backend/app/services/portfolio/review_engine.py`](../backend/app/services/portfolio/review_engine.py) parses rows, enriches prices (tiered), applies rules, detects option rolls from trades, auto-carries rationale when appropriate, and writes:
   - `portfolio/latest.json`
   - `portfolio/snapshots/<timestamp>.json`
   - `portfolio/reviews/<YYYY-MM-DD>.json` + `.md`
3. **Rationale** — Durable rationale lives in Postgres (`portfolio_rationales`) and is mirrored to `portfolio/rationales/<position_key>.json`. Rolls append a `portfolio_roll_events` row; undo removes the copied rationale on the new leg.

## OpenClaw cron

- In-repo script (copy to the gateway workspace if needed): [`scripts/openclaw-portfolio-sync.mjs`](../scripts/openclaw-portfolio-sync.mjs)
- Requires **`MISSION_CONTROL_TOKEN`** (org-admin bearer) and **`MISSION_CONTROL_BASE_URL`** (Mission Control API).
- Optional: `PORTFOLIO_SHEET_ID`, `PORTFOLIO_POSITIONS_RANGE`, `PORTFOLIO_TRADES_RANGE`, `PORTFOLIO_GOOGLE_ACCOUNT`.

**Environment file (OpenClaw Docker):** define the variables above in `~/.openclaw-docker/.env` (for example `/home/scott/.openclaw-docker/.env`) so the gateway / cron job process inherits them when it runs `portfolio-sync.mjs`.

The `morning-portfolio-review` cron job should run that script; the agent reads `review_markdown_path` from the JSON stdout.

## API

| Method | Path | Purpose |
|--------|------|---------|
| `POST` | `/api/v1/portfolio/review/run` | Run engine (body: `positions_rows`, `trades_rows`, optional `generated_at`) |
| `GET` | `/api/v1/portfolio/roll-events?days=7` | List recent roll events |
| `POST` | `/api/v1/portfolio/roll-events/{event_id}/undo` | Undo auto-carry + dismiss event |
